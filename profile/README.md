# VpprOrganization

## 📌 Sobre
Organização focada em desenvolvimento de soluções escaláveis, com ênfase em backend, integrações e automação de processos.

---

## 🧩 Stack principal
- Node.js
- PHP
- JavaScript
- APIs REST
- Integrações externas

---

## 🚀 Projetos
- **vpprnfse** → Emissão e gestão de NFS-e
- **Agilith2** → Backend PHP
- **Site-Vppr** → Frontend institucional
- **SolucoesVppr** → Serviços e integrações

## ⚙️ Padrões de desenvolvimento

### Pull Requests
- Uso obrigatório de template
- Classificação por tipo (feature, bugfix, refactor)
- Definição de impacto e risco

### Código
- Separação de responsabilidades
- Evitar acoplamento
- Priorizar legibilidade
